Notes About Transformations:

  1. It doesn’t work with inline-block elements
  2. When using an inline block element (span, a, select, textarea, label etc.) use display: inline-block to fix transform issue
  3. Transform property can contain plenty of methods and the methods are applied in the order they appear from left to right

Left-Right (TranslateX)

-X value = left

X value = right

Up-Down (TranslateY)

-Y value = up

Y value = down

Rotation

The rotate() function rotates an element around the point of origin by a specified angle value. As with scale, by default the point of origin is the element’s center. Generally, angles are declared in degrees, with positive degrees moving clockwise and negative moving counterclockwise.

Skew

The skew(x,y) function specifies a skew along the x and y axes. As you’d expect, the x specifies the skew on the x axis, and the y specifies the skew on the y axis. If the second parameter is omitted, the skew will only occur on the x axis.

Transition Functions

  • transition-delay: specifies a delay (in seconds) for transition effect
  • transition-duration: specifies how many seconds or milliseconds a transition effect takes to complete
  • transition-property: specifies the name of the CSS property the transition effect is for
  • transition-timing-function: Specifies the speed curve of the transition effect

Transition Shorthand Syntax

transition: property || duration || timing-function || delay [, ...];

Transition Timing Functions

  • ease: slow start, speeds up, ends quickly
  • linear: animates at a constant speed
  • ease-in: slow start, speeds up, stops abruptly
  • ease-out: full speed start then slows progressively as it reaches the conclusion of the transition.
  • ease-in-out: similar to ease, but accelerates more sharply at the beginning
  • cubic-bezier(n,n,n,n): lets user define their own values in a cubic-bezier function
